Useless API documentation

Endpoints

If you go to most of these and dont provide any arguments you will be redirected to the same endpoint but with example arguments

/welcomecard

Builds a welcome card image with a customizable background avatar and text.

/wanted

Makes a wanted poster with the specified avatar.

/sharpen

Applies a sharpening filter to the specified image (ImageFilter.SHARPEN)

/smooth

Smoothes out the image with PIL's smooth filter (ImageFilter.EDGE_ENHANCE)

/edge_enhance

Applies PIL's edge enhance filter (ImageFilter.EDGE_ENHANCE)

/contour

Applies PIL's contour filter (ImageFilter.CONTOUR)

/find_edges

Applies PIL's find_edges filter (ImageFilter.FIND_EDGES)

/random_math

Generates a random math operation

/random_word

Generates a random word

/random_sentence

Generates a random sentence with random words and the specified length (in words)
